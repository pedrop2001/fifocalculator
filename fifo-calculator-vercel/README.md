# FIFO Money Tools

A Vite + React + Tailwind website for FIFO pay, tax, overtime, super, and savings calculators.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

Import this project/repository into Vercel. The default Vercel settings should work:

- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist
